# Go-RPC

RPC server implementation in go. Simulating a cafeteria. Adding a Coffee, Getting all coffees.

## Structure

```
├── Readme.md
├── client
│   ├── main.go
│   └── makefile
├── go.mod
└── server
    ├── main.go
    └── makefile
```

Use make dev to run the server and client, to build use make build.
